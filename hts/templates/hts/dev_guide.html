개발자 가이드 (API 명세서) 메인으로 돌아가기

HTS 시스템 연동을 위한 RESTful API 명세서 및 Postman 테스트 예제입니다. 기본 도메인은 http://spiritlink.uk (또는 로컬 http://127.0.0.1:8000)을 사용합니다.

공통 응답 포맷: 모든 API는 일관된 JSON 구조(status, message, data/errors)를 반환합니다.

1. 주식 종목 전체 조회

GET /api/stocks/

시스템에 등록된 모든 주식 종목의 정보를 조회합니다.

Postman 테스트 (응답 예시)

{
    "status": "success",
    "message": "Stock list retrieved successfully",
    "data": [
        {
            "id": 1,
            "symbol": "005930",
            "name": "삼성전자",
            "market": "KR"
        }
    ]
}

2. 특정 종목의 기간별 주가 조회

GET /api/prices/symbol/range/

특정 주식(symbol)의 지정된 시간 범위 내 가격 데이터를 조회합니다. (차트 그리기 용도로 주로 사용됨)

Query Parameter 설명 및 예시
symbol 조회할 종목 코드 (예: 005930)
start_time 조회 시작 시간 (ISO 8601 포맷, 예: 2024-02-01T00:00:00Z)
end_time 조회 종료 시간 (ISO 8601 포맷, 예: 2024-03-01T23:59:59Z)

Postman 테스트 (URL 예시)

http://spiritlink.uk/api/prices/symbol/range/?symbol=005930&start_time=2024-02-01T00:00:00Z&end_time=2024-03-01T23:59:59Z
